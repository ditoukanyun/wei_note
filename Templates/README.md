## 🛠️ 如何使用模板

### 方法 1：Templater 插件（推荐）
1. 安装并启用 Templater 插件
2. 设置模板文件夹路径为 `Templates`
3. 使用快捷键或命令面板插入模板

### 方法 2：手动复制
1. 打开模板文件
2. 复制全部内容
3. 粘贴到新笔记中
4. 修改标题和内容

### 方法 3：模板按钮
在任意笔记中添加：
```markdown
[[Templates/项目模板|➕ 新建项目]]
[[Templates/技术笔记模板|➕ 新建学习笔记]]
[[Templates/每日复盘模板|➕ 今日复盘]]
```

## 📝 创建新模板

如果你需要新模板：

1. 在此文件夹新建 `.md` 文件
2. 使用 Obsidian 支持的模板语法
3. 常用变量：
   - `{{title}}` - 笔记标题
   - `{{date:YYYY-MM-DD}}` - 当前日期
   - `{{time:HH:mm}}` - 当前时间
