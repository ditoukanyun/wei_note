# 📂 Archives - 归档

## 定义
**归档**是已完成的项目、不再活跃的内容或暂时搁置的材料。

## 特点
- 📦 已完成或暂停
- 🔒 只读，不再修改
- 🔍 备查，可搜索
- 📜 历史记录

## 子分类

### 📁 已完成项目
已完成的项目，保留完整记录。

### 📁 旧笔记
过时的笔记或不再需要的参考材料。

### 📁 废弃想法
放弃的项目想法或计划。

## 归档规则

### 何时归档
- ✅ 项目已交付完成
- ✅ 学习项目已达到目标
- ✅ 内容已过时
- ✅ 暂时不再需要

### 归档步骤
1. 确认项目/内容已完成或不再需要
2. 添加最终复盘（如果是项目）
3. 移动到此文件夹的相应子目录
4. 更新所有相关链接

### 命名规范
```
[年份]_[原分类]_[原名]
```
例：
- `2024_工作项目_React组件库重构`
- `2024_学习项目_30天TypeScript`
