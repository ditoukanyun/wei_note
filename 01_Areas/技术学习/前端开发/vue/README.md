# Vue 框架系列

> Vue 是一套用于构建用户界面的**渐进式框架**。与其他大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。
