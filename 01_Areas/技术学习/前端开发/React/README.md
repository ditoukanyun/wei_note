# React 学习笔记 MOC

React 是用 JavaScript 构建快速响应的大型 Web 应用程序的首选方式。本知识库包含 React 核心概念、性能优化和源码解析等内容。

## 核心特性

- **声明式**：React 使创建交互式 UI 变得轻而易举。为你应用的每一个状态设计简洁的视图
- **组件化**：创建拥有各自状态的组件，再由这些组件构成更加复杂的 UI
