# 📂 Projects - 项目

## 定义
**项目**是有明确目标、可交付成果和截止日期的短期努力。

## 特点
- ✅ 有明确的开始和结束
- ✅ 有具体的交付成果
- ✅ 有截止日期
- ✅ 完成后可以归档

## 子分类

### 📁 工作项目
工作中的具体任务和项目。

### 📁 学习项目
有明确目标的学习计划（如：30天学会React）。

### 📁 生活项目
生活中的具体事务（如：装修、健身挑战）。

## 使用规范

### 命名格式
```
项目名_简短描述
```
例：
- `React项目_组件库重构`
- `学习计划_30天TypeScript`
- `生活计划_搬家`

### 状态管理
- `⏸️ 暂停` - 暂时搁置
- `🔄 进行中` - 正在执行
- `✅ 已完成` - 已交付

### 归档规则
- 项目完成后，移动到 `03_Archives/已完成项目/`
- 保留完整项目记录和复盘
