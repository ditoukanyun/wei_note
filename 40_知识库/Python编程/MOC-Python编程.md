## 🎯 学习路线推荐

### 路线1：零基础入门
适合完全没有编程经验的初学者

```
00-导航-Python编程导航 → 00-学习计划-100天路线 → 01-基础-语法入门 
→ 01-基础-数据结构详解 → 01-基础-函数与模块 → 01-基础-面向对象编程
```

### 路线2：Web 开发工程师
目标是成为 Python Web 开发者

```
01-基础-语法入门 → 02-应用-数据库与SQL → 03-方向A-Web开发-Django全栈 
→ 03-方向A-工程化实践
```

### 路线3：数据分析师
目标是成为数据分析师/数据科学家

```
01-基础-语法入门 → 03-方向C-数据分析 → 03-方向C-机器学习
```

### 路线4：爬虫工程师
目标是成为网络爬虫工程师

```
01-基础-语法入门 → 02-应用-进阶与Linux → 03-方向B-网络爬虫
```

## 📊 学习进度检查清单

### 基础里程碑
- [ ] **环境搭建完成** - Python 安装与 IDE 配置
- [ ] **语法入门完成** - 能独立写简单程序
- [ ] **数据结构掌握** - 熟练使用列表、字典
- [ ] **函数编程掌握** - 理解函数与模块
- [ ] **OOP 理解** - 掌握面向对象编程

### 应用里程碑
- [ ] **文件操作熟练** - 能处理各种文件格式
- [ ] **数据库入门** - 能使用 SQL 操作数据库
- [ ] **开发环境搭建** - 掌握 UV 与虚拟环境

### 专业里程碑
- [ ] **方向选择** - 确定 Web/数据/爬虫方向
- [ ] **框架掌握** - 熟练使用 Django/NumPy/Scrapy
- [ ] **项目实战** - 完成至少一个完整项目
- [ ] **工程化实践** - 了解部署与 CI/CD
