## 4. 控制流

### 4.1 条件语句
```python
age = 18

if age < 13:
    print("儿童")
elif age < 20:
    print("青少年")  # 输出
else:
    print("成年人")

# 三元表达式
status = "成年" if age >= 18 else "未成年"
```

### 4.2 循环

#### for循环
```python
# 遍历列表
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(fruit)

# 遍历范围
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# 带索引遍历
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
```

#### while循环
```python
count = 0
while count < 5:
    print(count)
    count += 1
```

#### 循环控制
```python
for num in range(10):
    if num == 3:
        continue    # 跳过当前迭代
    if num == 7:
        break       # 终止循环
    print(num)
```

## 5. 数据结构

### 5.1 列表（List）
```python
# 创建列表
my_list = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# 常用操作
my_list.append(6)       # 添加元素
my_list.insert(0, 0)    # 插入元素
my_list.remove(3)       # 删除元素
del my_list[0]          # 删除索引处元素
my_list.pop()           # 删除并返回最后一个元素

# 切片
my_list[1:4]    # 索引1到3
my_list[:3]     # 开头到索引2
my_list[2:]     # 索引2到结尾
my_list[::2]    # 每隔一个取一个
my_list[::-1]   # 反转列表

# 列表方法
len(my_list)            # 长度
my_list.sort()          # 排序
my_list.reverse()       # 反转
my_list.count(2)        # 计数
```

### 5.2 元组（Tuple）
```python
# 元组是不可变的
my_tuple = (1, 2, 3)
single = (1,)  # 单元素元组需要逗号

# 解包
x, y, z = my_tuple
```

### 5.3 字典（Dictionary）
```python
# 创建字典
person = {
    "name": "张三",
    "age": 25,
    "city": "北京"
}

# 访问和修改
print(person["name"])       # 访问
person["age"] = 26          # 修改
person["job"] = "工程师"     # 添加

# 安全访问
person.get("phone", "未知")  # 带默认值的获取

# 遍历
for key in person:
    print(key, person[key])

for key, value in person.items():
    print(f"{key}: {value}")

# 常用方法
person.keys()       # 所有键
person.values()     # 所有值
person.items()      # 所有键值对
person.pop("age")   # 删除并返回值
```

### 5.4 集合（Set）
```python
# 创建集合
my_set = {1, 2, 3, 3, 3}  # 自动去重: {1, 2, 3}

# 集合操作
set1 = {1, 2, 3}
set2 = {3, 4, 5}

set1 | set2     # 并集: {1, 2, 3, 4, 5}
set1 & set2     # 交集: {3}
set1 - set2     # 差集: {1, 2}
set1 ^ set2     # 对称差集: {1, 2, 4, 5}
```

### 5.5 列表推导式
```python
# 基本形式
squares = [x**2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 带条件
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]

# 字典推导式
square_dict = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 集合推导式
square_set = {x**2 for x in range(100)}
```

## 6. 函数

### 6.1 函数定义
```python
def greet(name):
    """函数的文档字符串（docstring）"""
    return f"你好, {name}!"

# 调用
message = greet("张三")
print(message)
```

### 6.2 参数类型
```python
# 位置参数
def add(a, b):
    return a + b

# 默认参数
def power(base, exponent=2):
    return base ** exponent

# 关键字参数
power(base=2, exponent=3)

# 可变参数 *args
def sum_all(*args):
    return sum(args)

sum_all(1, 2, 3, 4)  # 10

# 关键字可变参数 **kwargs
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="张三", age=25)
```

### 6.3 返回值
```python
# 返回多个值（实际是返回元组）
def get_stats(numbers):
    return min(numbers), max(numbers), sum(numbers)/len(numbers)

min_val, max_val, avg = get_stats([1, 2, 3, 4, 5])
```

### 6.4 Lambda表达式
```python
# 简单函数
square = lambda x: x ** 2
print(square(5))  # 25

# 作为参数
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))
```

## 7. 文件操作

### 7.1 读写文件
```python
# 读取整个文件
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# 逐行读取
with open('file.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())

# 写入文件
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello, World!')

# 追加模式
with open('log.txt', 'a', encoding='utf-8') as f:
    f.write('New log entry\n')
```

### 7.2 文件路径
```python
import os

# 获取当前目录
current_dir = os.getcwd()

# 路径拼接
file_path = os.path.join('folder', 'subfolder', 'file.txt')

# 检查文件是否存在
os.path.exists('file.txt')

# 获取文件信息
os.path.getsize('file.txt')
os.path.isfile('file.txt')
os.path.isdir('folder')
```

## 8. 异常处理

### 8.1 基本结构
```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("不能除以零")
except Exception as e:
    print(f"其他错误: {e}")
else:
    print("没有异常时执行")
finally:
    print("无论有无异常都执行")
```

### 8.2 自定义异常
```python
class ValidationError(Exception):
    pass

def validate_age(age):
    if age < 0:
        raise ValidationError("年龄不能为负数")
    return age
```

## 10. 编码规范

### 10.1 命名规范
- 变量名：小写，下划线分隔 `user_name`
- 常量：大写，下划线分隔 `MAX_SIZE`
- 函数名：小写，下划线分隔 `calculate_total`
- 类名：驼峰命名 `UserAccount`

### 10.2 代码风格（PEP 8）
```python
# 缩进：4个空格
def my_function():
    if True:
        print("正确")

# 行长度：最多79字符
# 空行：函数之间2行，类方法之间1行

# 导入顺序
import os          # 标准库
import sys

import requests    # 第三方库

from mymodule import myfunc  # 本地模块
```
